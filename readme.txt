基于最优质量传输理论的点云骨架提取
：
1.main.m，程序入口，输入点云数据，输出曲线骨架

2.ComputeSkeleton.m，根据最优传输计划计算骨架点位置。输入点云坐标，采样点坐标，初始化质量，输出骨架点位置、最优传输方案OT、骨架点质量。
3.ot.m，计算最优传输方案函数。输入点云坐标和采样点坐标，输出最优传输方案OT

4.center.m，计算采样点坐标函数。输入点云坐标和传输方案OT，输出更新后的采样点坐标

5.showSkel.m，计算采样点连接关系函数。输入传输方案OT，输出连接关系矩阵

6.NewComputeSkeleton.m，根据初始连接关系，检测对称性，去环。输入点云、骨架点、连接关系、质量和传输方案OT，输出骨架点、新的连接关系、质量和传输方案OT。
7.smooth.m，平滑骨架分支。输入点云坐标，骨架分支和插入点个数，输出平滑后的骨架坐标

8.sinkhornTransport.m，熵约束法函数。
